# pimcore php 8.1-fpm-debug

## Latest version: 1.0

Build image: 
```
docker build --platform linux/amd64 -t ghcr.io/laurenz-glueck/pimcore-php-docker-images/pimcore-php/php-8-1-fpm-debug:1.0 .
```

You might want to use the `--no-cache` option for a fresh build without cache.

Push image:
```
docker push ghcr.io/laurenz-glueck/pimcore-php-docker-images/pimcore-php/php-8-1-fpm-debug:1.0
```
