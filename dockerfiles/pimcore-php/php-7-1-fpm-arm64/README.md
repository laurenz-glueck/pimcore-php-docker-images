# pimcore php 7.1-fpm arm64

## Latest version: 1.0

Build image: 
```
docker build --platform linux/arm64 -t ghcr.io/laurenz-glueck/pimcore-php-docker-images/pimcore-php/php-7-1-fpm-arm64:1.0 .
```

You might want to use the `--no-cache` option for a fresh build without cache.

Push image:
```
docker push ghcr.io/laurenz-glueck/pimcore-php-docker-images/pimcore-php/php-7-1-fpm-arm64:1.0
```
